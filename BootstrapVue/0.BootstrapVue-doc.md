# BootstrapVue

BootstrapVue is a component library that integrates **Bootstrap v4 CSS** with **Vue.js v2.6**. It provides:

- 85+ ready-to-use Vue components
- 45+ plugins & directives
- 1200+ icons
- Built-in support for responsive layout & accessibility (WAI-ARIA)

It implements Bootstrap’s grid and styles via Vue components, components becomes `<b-button>`, `<b-card>`, `<b-modal>`, etc., instead of raw HTML + CSS classes.

# Module 0 — Basic Overview

### Layout & Grid System

BootstrapVue uses Bootstrap v4’s grid with Vue components:

**Grid Basics**

```vue
<b-container>
  <b-row>
    <b-col>Column 1</b-col>
    <b-col>Column 2</b-col>
    <b-col>Column 3</b-col>
  </b-row>
</b-container>
```

Core parts:

- `<b-container>` — wraps content; use fluid for full width
- `<b-row>` — horizontal row of columns
- `<b-col>` — responsive column

Columns are based on a `12-unit grid` and adapt to breakpoints (`sm`, `md`, etc.).

### Components Overview

BootstrapVue has components for almost all Bootstrap UI scenarios. Some major ones:

#### UI & Layout

- **Buttons** — `<b-button>` variations
- **Cards** — `<b-card>` content containers
- **Navbar** — responsive nav bars
- **Layout** — grid, container helpers

#### Content & Feedback

- **Alerts** — `<b-alert>` contextual messages
- **Modals** — `<b-modal>` dialogs
- **Toasts** — toast notifications
- **Tooltips**/**Popovers** — helpers via components or directives

#### Forms & Inputs

- **Form controls** — `<b-form-input>`, `<b-form-select>`
- **Checkboxes**/**Radios** — styled groups
- **Datepicker**/**Timepicker** — interactive inputs

#### Data Display

- **Tables** — `<b-table>` with pagination/sorting
- **Tabs** — `<b-tabs>` for sections
- **Carousel** — slideshows

### Directives

BootstrapVue provides directives when full components aren't needed:

- **v-b-tooltip** — adds tooltips
- **v-b-popover** — popovers
- **v-b-toggle** — toggles visibility
- **v-b-scrollspy** — scrollspy context

Directives work like Vue’s `v-model`, `v-show` etc., and provide behavior enhancements to any element.

### Icons

BootstrapVue includes a comprehensive **SVG icons set** which can be used directly in templates with `<b-icon>` or imported individually.

Icon size, color, and accessibility attributes also can be customized.
